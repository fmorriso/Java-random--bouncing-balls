# Java Random Bouncing Balls
An example of using Java Swing to generate a random number of bouncing balls in a square JFrame that is scaled to a percentage of the device width and height.

## Tools Used

| Tool     |  Version |
|:---------|---------:|
| Java     | 24.0.0.0 |
| IntelliJ | 2024.3.5 |
| VSCode   |   1.98.2 |
